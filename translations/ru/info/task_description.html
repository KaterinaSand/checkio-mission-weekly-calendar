Давайте составим недельный календарь!

Вам даны четыре целых числа (год, месяц, день и первый день недели [0: Понедельник ... 6: Воскресенье])
Вам нужно вернуть неделю в виде перечня дат, включая заданную, как список (или итерируемое).
Она должна начинаться с первого дня недели.

Пример: 

list(weekly_calendar(2020, 1, 1, 0)) == [30, 31, 1, 2, 3, 4, 5]
list(weekly_calendar(2020, 9, 20, 6)) == [20, 21, 22, 23, 24, 25, 26]
list(weekly_calendar(2020, 9, 30, 0)) == [28, 29, 30, 1, 2, 3, 4]

Входные данные: четыре целых числа (год, месяц, день и первый день недели).

Выходные данные: список (или итерируемое) из семи целых чисел.

Предусловия:

  • datetime.date(1, 1, 1) ≤ datetime.date(year, month, day) ≤ datetime.date(9999, 12, 31)
  • 0 ≤ первый день недели ≤ 6
